## 🎯 О проекте

**FitClub** — это современное одностраничное веб-приложение (SPA) для управления фитнес-клубом. Приложение реализует полный цикл взаимодействия: от регистрации клиента и покупки абонемента до записи на тренировки, выбора персонального тренера и визуализации проблемных зон тела в интерактивном 3D-режиме.

### Ключевые особенности

| Возможность | Описание |
|:---|:---|
| 🔐 **Ролевая система** | 3 роли: клиент, тренер, администратор — каждая со своим интерфейсом |
| 🏆 **Каталог тренеров** | 10 профессиональных тренеров с профилями, достижениями и расписанием |
| 🧬 **3D-конструктор тела** | Интерактивная Three.js модель с выбором проблемных зон и персональными рекомендациями |
| 💳 **Система оплаты** | Форма оплаты с валидацией по алгоритму Луна, проверкой срока действия и CVV |
| 📅 **Онлайн-запись** | Запись на групповые и индивидуальные тренировки в реальном времени |
| 📊 **Аналитика** | Дашборды со статистикой посещаемости, выручки и загрузки зала |
| 🐳 **Контейнеризация** | Docker-образ на базе Nginx Alpine с health-check и gzip-сжатием |
| ⚙️ **CI/CD** | GitHub Actions: тесты → линтинг → Docker-сборка → деплой на GitHub Pages |

## 🚀 Функциональность

### 👤 Роль: Клиент

- 📊 **Дашборд** — персональная панель с информацией об абонементе, ближайших тренировках и статистике посещений
- 🎫 **Абонементы** — покупка из 4 тарифов (разовое / месячный / квартальный / годовой) с формой оплаты банковской картой
- 📅 **Расписание** — просмотр всех тренировок с индикатором заполненности, запись и отмена в один клик
- 🏆 **Тренеры** — каталог из 10 тренеров с фильтрацией, профилями, отзывами, достижениями и онлайн-записью на свободные слоты
- 🧬 **3D-конструктор тела** — выбор пола, возраста, процента жира (5–50%) и проблемных зон; получение персональных программ тренировок и подбор тренера
- 📋 **История** — таблица всех посещений со статусами
- 👤 **Профиль** — просмотр личной информации
- 🎉 **Скидка для новичков** — пробная тренировка с любым тренером за 200 ₽

### 🏅 Роль: Тренер

- 📊 **Кабинет** — количество занятий, записанных клиентов, опыт и направления
- 📅 **Расписание** — просмотр своих тренировок с загрузкой по каждой
- ✅ **Посещаемость** — отметка присутствия/отсутствия клиентов на каждом занятии
- 🧬 **3D-конструктор** — доступ к интерактивному конструктору тела

### ⚙️ Роль: Администратор

- 📊 **Панель управления** — общие KPI: количество клиентов, активные абонементы, тренировки, выручка
- 📈 **Отчёты** — визуальные графики посещаемости по дням недели и продаж абонементов
- 📅 **Расписание** — таблица всех тренировок с тренерами, датами, местами и статусами
- 🎫 **Абонементы** — управление абонементами всех клиентов
- 👥 **Пользователи** — полный список всех пользователей системы с ролями

## 📦 Модули приложения

### 🧬 3D-конструктор тела

Интерактивный модуль на базе **Three.js** с процедурной генерацией 3D-модели человеческого тела:

- **Выбор пола** — мужская/женская модель с анатомическими различиями (ширина плеч, бёдер, пропорции)
- **Процент жира** — слайдер 5–50% динамически изменяет объёмы тела (живот, бока, руки, ноги)
- **10 проблемных зон** — живот, бока, руки (трицепс/бицепс), ноги, грудь, спина, ягодицы, плечи, икры
- **Подсветка зон** — выбранные зоны подсвечиваются красным на 3D-модели
- **Персональные рекомендации** — 3–4 программы на каждую зону с длительностью и частотой
- **Комбо-программы** — 8 специальных связок (Ноги+Икры, Живот+Бока, Грудь+Плечи и др.)
- **Подбор тренера** — автоматический подбор тренеров по специализации в выбранных зонах
- **Истории успеха** — реальные кейсы трансформации по выбранным зонам
- **Автоматическое вращение** — плавная 3D-анимация модели

### 🏆 Система тренеров

- **10 профессиональных тренеров** разных специализаций (силовые, йога, кроссфит, пилатес, бокс, танцы, бодибилдинг, TRX, плавание, pole dance)
- **Детальные профили** — ФИО, возраст, опыт, цена, рейтинг, биография, достижения, методики
- **Отзывы клиентов** — реальные отзывы с рейтингами
- **Недельное расписание** — автоматически генерируемое расписание на 7 дней
- **Онлайн-запись** — выбор свободного слота → модалка оплаты → подтверждение
- **Пробная тренировка** — скидка для новых клиентов (200 ₽ вместо полной цены)

### 💳 Система оплаты

- **Валидация номера карты** — алгоритм Луна для проверки корректности 16-значного номера
- **Проверка срока действия** — формат ММ/ГГ, проверка что карта не просрочена
- **CVV-проверка** — 3 цифры, маскирование ввода
- **Имя владельца** — только буквы, минимум 3 символа
- **Автоформатирование** — номер карты (XXXX XXXX XXXX XXXX) и срок (ММ/ГГ)

### 📊 Виджет загрузки зала

- **Индикатор в реальном времени** — количество людей в зале (0–50)
- **Цветовая индикация** — зелёный (≤10), жёлтый (11–25), красный (>25)
- **Симуляция** — обновление каждые 30 секунд

## 🐳 Docker

### Сборка и запуск

```bash
# Сборка Docker-образа
docker build -t fitclub:latest .

# Запуск контейнера
docker run -d -p 8080:80 --name fitclub fitclub:latest

# Или через Docker Compose
docker-compose up -d
```

Приложение будет доступно по адресу: **http://localhost:8080**

### Makefile команды

```bash
make help       # Показать доступные команды
make build      # Собрать Docker-образ
make run        # Запустить в Docker
make stop       # Остановить контейнер
make test       # Запустить тесты
make lint       # Проверить код
make clean      # Очистить Docker-образы
```

### Dockerfile

```dockerfile
FROM nginx:alpine
COPY app/ /usr/share/nginx/html/
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
HEALTHCHECK --interval=30s --timeout=3s --retries=3 \
  CMD wget -qO- http://localhost/ || exit 1
CMD ["nginx", "-g", "daemon off;"]
```

## 🔒 Безопасность

### Реализованные меры

| Мера | Реализация |
|:---|:---|
| **XSS-защита** | Функция `escapeHtml()` экранирует `< > & " '` во всех пользовательских вводах |
| **Валидация ввода** | Клиентская валидация email, телефона, пароля, номера карты (алгоритм Луна) |
| **Security Headers** | `X-Frame-Options`, `X-Content-Type-Options`, `X-XSS-Protection`, `CSP`, `Referrer-Policy` |
| **CSP** | `Content-Security-Policy` ограничивает источники скриптов, стилей и шрифтов |
| **Gzip-сжатие** | Nginx сжимает CSS/JS/JSON для уменьшения трафика |
| **Кэширование** | Статические ресурсы кэшируются на 7 дней (`Cache-Control: public, immutable`) |
| **Скрытые файлы** | Nginx блокирует доступ к `.git`, `.env` и другим скрытым файлам |
| **Docker healthcheck** | Контейнер проверяет доступность каждые 30с |
| **Error handling** | Все критические функции обёрнуты в `try/catch` |
| **Регистрация** | Валидация ФИО (≥5 символов, ≥2 слова), email (уникальность), телефон, пароль (≥6 символов, ≥1 цифра) |
