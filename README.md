# 🏋️ FitClub — Сервис фитнес-клуба (v3.0 Engineering)

> Веб-приложение для управления абонементами, записи на тренировки и учёта посещаемости фитнес-клуба.  
> **УП.03** — инженерная версия с Docker, CI/CD, тестами и безопасностью.

---

## 📋 О проекте

Проект представляет собой одностраничное веб-приложение (SPA) для автоматизации ключевых процессов фитнес-клуба. Система обеспечивает взаимодействие трёх ролей: **клиентов**, **тренеров** и **администраторов**.

| Роль | Возможности |
|:---|:---|
| **Клиент** | Запись на тренировки, покупка абонемента, просмотр остатка и истории |
| **Тренер** | Просмотр расписания, электронная отметка присутствия |
| **Администратор** | Управление расписанием, абонементами, статистика и отчёты |

## 🚀 Как запустить

### Вариант 1: Прямой запуск (Windows)
```
start.bat
```

### Вариант 2: Docker
```bash
docker-compose up -d
# Приложение: http://localhost:8080
```

### Вариант 3: Открыть файл
Откройте `app/index.html` в браузере.
