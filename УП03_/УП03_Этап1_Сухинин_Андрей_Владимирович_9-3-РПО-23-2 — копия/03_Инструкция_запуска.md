# Инструкция запуска проекта

1. **Клонирование репозитория**
   ```bash
   git clone <URL_из_repo_link.txt>
   cd <папка_проекта>
   ```
2. **Установка зависимостей** (если проект использует npm) 
   ```bash
   npm install
   ```
3. **Подготовка окружения**
   - Скопировать файл `.env.example` в `.env` и заполнить свои значения.
4. **Запуск проекта**
   ```bash
   npm run dev
   ```
   Откройте браузер по адресу `http://localhost:3000`.
5. **Проверка работы**
   - Убедитесь, что в терминале нет ошибок.
   - Сравните скриншоты в папке `screenshots/` с текущим выводом.
