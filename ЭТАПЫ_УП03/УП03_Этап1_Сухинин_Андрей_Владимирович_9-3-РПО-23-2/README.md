## Как запустить проект локально

```bash
# Клонировать репозиторий
git clone https://github.com/valtureso788/Fitness-03
cd Fitness-03

# Установить зависимости (если есть)
npm install

# Запуск в режиме разработки
npm run dev
```

> При необходимости скопируйте `.env.example` в `.env` и отредактируйте переменные.
