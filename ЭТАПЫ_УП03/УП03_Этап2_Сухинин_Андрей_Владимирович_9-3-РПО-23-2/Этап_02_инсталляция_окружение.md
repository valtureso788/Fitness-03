## 3. Docker-контейнеризация

### 3.1. Dockerfile

Используется `nginx:alpine` — легковесный веб-сервер для раздачи статических файлов.

```dockerfile
FROM nginx:alpine
LABEL maintainer="FitClub Engineering Team"
LABEL version="3.0"
LABEL description="FitClub — Сервис фитнес-клуба"

COPY app/ /usr/share/nginx/html/
COPY nginx.conf /etc/nginx/conf.d/default.conf

EXPOSE 80

HEALTHCHECK --interval=30s --timeout=3s --retries=3 \
  CMD wget -qO- http://localhost/ || exit 1

CMD ["nginx", "-g", "daemon off;"]
```

**Обоснование выбора:**
- `nginx:alpine` — образ ~5 МБ, минимальная поверхность атаки
- `HEALTHCHECK` — автоматическая проверка состояния контейнера каждые 30 секунд
- Собственный `nginx.conf` — заголовки безопасности, gzip-сжатие и кэширование

### 3.2. Docker Compose

```yaml
version: '3.8'
services:
  fitclub:
    build: .
    container_name: fitclub-app
    ports:
      - "8080:80"
    restart: unless-stopped
    healthcheck:
      test: ["CMD", "wget", "-qO-", "http://localhost/"]
      interval: 30s
      timeout: 3s
      retries: 3
```

## 4. Скрипты запуска

### 4.1. Windows — `start.bat`

Скрипт автоматически определяет доступный способ запуска:
1. Python → `python -m http.server 8000` (порт 8000)
2. Node.js → `npx serve app -l 3000` (порт 3000)
3. Fallback → открытие `index.html` напрямую в браузере

### 4.2. Windows — `test.bat`

Скрипт запуска автоматизированных тестов:
- Проверяет наличие Node.js
- Запускает `node tests\run-tests.js`
- Выводит результаты 21 теста

### 4.3. Makefile (Linux/macOS)

| Команда | Описание |
|:---|:---|
| `make help` | Справка по всем командам |
| `make build` | Сборка Docker-образа |
| `make run` | Запуск контейнера (http://localhost:8080) |
| `make stop` | Остановка контейнера |
| `make test` | Запуск 21 теста |
| `make lint` | Проверка HTML/CSS/JS |
| `make clean` | Очистка Docker-образов |

## 5. Конфигурация Nginx

Файл `nginx.conf` (36 строк) обеспечивает:

| Функция | Реализация |
|:---|:---|
| **Gzip-сжатие** | text/plain, text/css, application/javascript, application/json |
| **Заголовки безопасности** | X-Frame-Options, X-Content-Type-Options, X-XSS-Protection, Referrer-Policy, CSP |
| **Кэширование статики** | CSS/JS/изображения на 7 дней с `Cache-Control: public, immutable` |
| **SPA fallback** | `try_files $uri $uri/ /index.html` |
| **Блокировка скрытых файлов** | `location ~ /\. { deny all; return 404; }` |
